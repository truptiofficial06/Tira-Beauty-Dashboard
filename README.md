# Tira-Beauty-Dashboard
💄 Tira Beauty Performance Dashboard

1.📌 Project Overview
The dashboard provides a detailed analysis of revenue, product performance, customer ratings, and category trends. It helps understand how different brands and product categories contribute to overall business performance.

2.🎯 Objectives
Analyze total revenue and sales performance
Understand category-wise contribution
Evaluate customer ratings and product distribution
Track revenue trends over time
Enable interactive data exploration using filters

3.📊 Key Features
💰 Total Revenue KPI to track overall earnings
📦 Total Products Count
⭐ Average Rating Analysis
🎯 Average Discount Insights
🛍️ Category-wise Revenue Breakdown
🏷️ Brand-wise Revenue Analysis
📈 Revenue Trend (Quarter & Month)
🔎 Interactive Slicers (Brand, Category, Price, Rating)

4.🧠 Insights Gained
Identified top-performing category (e.g., Skincare)
Observed variation in revenue across brands
Analyzed how discounts influence purchasing behavior
Compared product ratings distribution
Tracked revenue fluctuations over time

5.🛠️ Tools & Technologies
Microsoft Power BI
Data Visualization
Data Cleaning & Transformation
